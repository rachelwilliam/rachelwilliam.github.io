Website Evaluations

CRAP Principles:
  • Contrast
  • Repetition
  • Alignment
  • Proximity

Good Example: Dr. Pepper

This website has information about Dr. Pepper, a carbonated drink.
  • Contrast: Good contrast using black and various shades of red with white text.
  • Repetition: The home page is very minimalistic. The position of the links alternate.
  • Alignment: There are three distinct sections that have the same size and alignment.
  • Proximity: The different sections are equidistant.

Bad Example: BBC News

This website shares the news.
  • Contrast: Good contrast using red, gray, and black. The images used also provide a lot of contrast against the white background.
  • Repetition: There are a lot of pictures of different sizes on the website and some of them are not using the same alignment which makes it a bit unpleasant to look at.
  • Alignment: Some of the news articles are different sizes skewing the alignment.
  • Proximity: There are a lot of articles all really close together.
